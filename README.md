# 基于群智能算法的TDOA定位算法仿真软件设计开发
## 项目背景及意义
无线定位技术是具有广泛应用前景的一种动关键技术。该技术能够有效利用现有蜂窝移动通信网络,通过对到达时间(TOA)、到达时间差(TDOA)等特征参数的测量和计算，利用定位算法来实现对目标用户的定位。
多个TDOA测量值构成双曲线方程组是非线性的，求解非线性的双曲线方程组,计算量大、时间长。目前有许多求解方程组的方法被提出，其中应用最多的是Chan算法和Taylor算法，但是Chan算法只有在测距误差较小的情况下才能获得较好的定位结果，Taylor算法则是一种循环迭代的递归算法，求解精度低。 
本项目利用粒子群优化算法解决TDOA定位估计中遇到的非线性最优化问题，针对TDOA定位方式,该算法首先初始化一个随机粒子群,然后根据适应度值更新粒子速度和位置,通过迭代搜索最佳坐标.仿真结果表明,在参数设定合理的情况下,该算法性能稳定,能找到逼近全局最优点的解,相对于其他算法精度更高。
## 项目内容
无线定位技术能够有效利用现有蜂窝移动通信网路，通过对到达时间差（TDOA）等特征参数的测量和计算实现对目标用户的定位。多个TDOA测量值构成双曲线方程组是非线性的，为保证定位实时性，需要解决求解方程组计算量大、时间长的问题。
## 实现思路
* 关于非线性多参数目标函数优化的问题，粒子群优化方法相比于遗传算法速度更快，相比于最速下降法，更容易找到全局最优解，因此，引入粒子群优化算法对声源位置进行搜索。
* 在建立TDOA双曲线数学模型并以粒子群算法求解模型条件下，采用蒙特卡洛仿真对算法的求解精度加以验证。该仿真方法能够比较逼真地描述具有随机性质的事物的特点及物理实验过程。受几何条件限制小。收敛速度与问题的维数无关。误差容易确定。程序结构简单，易于实现。
## 实现方法
* 研究分析TDOA定位原理并利用matlab建立TDOA定位数学模型；
* 基于多种群智能算法实现目标位置估计，采用C#进行仿真软件开发；
* 依托蒙特卡洛仿真得出在随机噪声环境下不同算法的求解误差，寻找最优解决方案；
